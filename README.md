PolutaDB（ぽるうたデータベース）は、ホロライブ所属の VTuber 尾丸ポルカさんの歌を検索することができる Web アプリです。
https://www.polutadb.com で公開されています。

## Requirement

- Node.js (v18)
- yarn

## Usage

### To Serve in Development Mode

```
$ yarn install
$ yarn dev
```

### To Serve in Production Mode

```
$ yarn install
$ yarn build
$ yarn start
```
